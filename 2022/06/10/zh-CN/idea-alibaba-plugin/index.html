IntelliJ IDEA 必备插件之阿里巴巴代码规范插件

前言

作为一个码农程序猿，每个人都想写出标准、规范且高质量的代码。但是规范的标准是什么呢？什么样的代码才是高质量的代码？估计每个人心中的标准也不是完全一致的。那么我们的评判标准如何做到统一，如何将写出来的代码最大程度的接近公认的规范呢?这时阿里巴巴代码规范插件应运而生。

阿里巴巴 Java 代码规范插件是阿里巴巴在 2017 年 10 月 14 日在杭州云栖大会首发，插件实现了《阿里巴巴 Java 开发手册》中的 53 条规则，提供了实时检测和部分规则的一键修复功能，后续应该会更加完善。

安装

插件可以使用在线安装和离线安装两种方式，根据个人偏好和需要任选一种安装即可：

1.在线安装

  • 打开 IDEA，并依次打开 File–>Settings–>Plugins，如下图：

  • 在搜索框输入 alibaba 即可看到 Alibaba Java Code Guidelines 插件，点击 Install 进行安装

  • 重启 IDE 生效

2.离线安装

  • 去官网 plugins 下载对应的插件离线安装包（需要先接受下载协议才能下载哦）

下载地址：https://plugins.jetbrains.com/plugin/10046-alibaba-java-coding-guidelines/versions

  • 依然是 File–>Settings–>Plugins,点小齿轮选择 Install Plugin from Disk，找到我们上一步下载的插件安装，并重启生效

使用

插件的使用的有两种场景：

全量扫描

如果我们在安装插件之前，已经写好了项目或者某个类，是不是就没办法检查代码是否符合规范了呢？别慌，只要我们选中想要审查的项目或者类，进行编码规范扫描，一样能达到我们想要的效果。

扫描结果如下：

扫描结果说明：

Blocker：阻断，最严重的问题，必须改！红色提醒

Critical：推荐，不是很严重，但也不规范，要改！橙黄提醒

Major：参考，一般，可改可不改，建议还是改！浅黄提醒

扫出来之后每个都有详细的结果说明，怎么改，为啥改，以及代码具体位置

中文乱码解决

最后如果发现菜单栏的阿里插件出现中文乱码,是因为菜单栏的字体不持支中文，可以 Appearance&Behavior -> Appearance -> UI Options -> Name 里面设置成微软雅黑（microsoft yahei light）或者自己喜欢的其他字体哦

学完了插件的安装及使用，赶紧给你的 IDEA 安装上吧，相信你的代码质量会更上一个台阶!😄
